# MFIPPA Compliance Checker

A web-based compliance checking tool for Ontario's Municipal Freedom of Information and Protection of Privacy Act (MFIPPA).

## Features
- Automated detection of personal information
- Real-time compliance analysis
- MFIPPA section citations
- Color-coded compliance reports

## Technologies
- Python
- Streamlit
- Pandas
- Regular Expressions

## How to Use
1. Visit the live app: [Link will be added]
2. Paste document text or upload a file
3. Review the automated compliance report

```bash
pip install -r requirements.txt
streamlit run app.py
```
